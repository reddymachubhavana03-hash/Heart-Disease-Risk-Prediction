# Heart Disease Risk Prediction using Machine Learning

## Project Overview
This project focuses on predicting the risk of cardiovascular disease using machine learning techniques and clinical patient data.

The project was developed using the Cleveland Heart Disease dataset and implements multiple classification algorithms to analyze and predict heart disease risk.

---

## Objectives
- Perform exploratory data analysis on healthcare data
- Build machine learning classification models
- Compare model performance
- Identify important clinical predictors
- Develop a simple deployment-ready prediction interface using Streamlit

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
- Jupyter Notebook

---

## Dataset
Cleveland Heart Disease Dataset  
Source: UCI Machine Learning Repository

Features include:
- Age
- Chest pain type
- Resting blood pressure
- Cholesterol
- Maximum heart rate
- ECG-related parameters
- Exercise-induced angina
- ST depression

---

## Machine Learning Models Implemented
### Logistic Regression
- Accuracy: ~79%

### Random Forest Classifier
- Accuracy: ~98.5%

---

## Key Findings
The most influential predictors identified by the Random Forest model were:
1. Chest pain type
2. Number of major vessels
3. Maximum heart rate achieved
4. ST depression (Oldpeak)
